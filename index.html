    What is LaunchQuery?

    LaunchQuery is a smart search tool that lets you quickly search across multiple websites using shortcuts (called "bangs").

    How to use it

    Simply type your search query along with a bang command (e.g., "cats !yt" to search YouTube for cats).

    The "!" can appear anywhere in your search - at the beginning, middle, or end.

    Examples

    • lebron james !w - Search for Lebron James on Wikipedia
    • !g programming tutorials - Search Google for programming tutorials
    • new movies !imdb - Search IMDB for new movies

    About the Developer

    LaunchQuery was created to make web searching faster and more efficient. It was inspired by DuckDuckGo's bang syntax but designed to work with any search engine at much faster speeds.
